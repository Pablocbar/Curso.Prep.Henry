&& || !

    Estos son operadores logicos, y cada uno cumple una funcion

    && - En el caso de && (and) lo que hace es evaluar  expreciones, 
         siempre que todas las expresiones sean true devolvera true, si alguna 
         o todas las expresionces son flase devolvera false
         si la expresion A es true && (Y) la expresion B es true el resultado sera true, 
         si alguno o las dos son false siempre sera false

    || - A este operador de lo conoce como "OR" u "Ã“", lo que haces que evaluar dos expresiones
         siempre que alguna de las dos expresiones sea true devolvera true  
         ejemplo
         si A es True o B es true devolvera true pero si A es true y B es False devolvera True, 
         lo que haces es buscar una de las expresiones sea true siempre que una sea true devolvera true
